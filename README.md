# Font Changer

**Font Changer** is a web-based Unicode text styler that converts normal text into multiple fancy fonts.  
It’s perfect for social media posts, messages, or anywhere you want text to stand out.

---

## Demo
Open `index.html` in your browser and type some text to see it styled instantly.  

Features include:
- Normal, Bold, Italic, Bold Italic
- Script, Bold Script
- Fraktur, Bold Fraktur
- Monospace, Double-Struck
- Retro (Gothic-ish)
- Old English (medieval style)

Copy your styled text with the **Copy** button.

---

## Getting Started

Clone the repository:

```bash
git clone https://github.com/username/Font-Changer.git
```

Open `index.html` in any modern browser (Chrome, Firefox, Edge).
Optional: serve with local server:
```bash
# Using Python
python -m http.server 8000
```
Then open using `http://localhost:8000`
